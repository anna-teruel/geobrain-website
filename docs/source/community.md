## Contributing

We welcome and encourage contributions of any form, whether it's fixing a bug, developing a new feature, or improving documentation.

* **Found a bug or have a feature request?** Open an issue on [GitHub](https://github.com/anna-teruel/geobrain/issues).
* **Want to contribute code or docs?** Fork the [repository](https://github.com/anna-teruel/geobrain) and open a pull request.
